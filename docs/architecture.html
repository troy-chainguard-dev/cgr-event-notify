What You Get Alerted On
🔴

New CVE Discovered

A vulnerability appears in an image's report that wasn't there before. Includes severity, affected package, version, and fix status.

✅

CVE Resolved

A vulnerability that was previously present is no longer in the report — patched in a rebuilt image or marked as false positive.

🟡

Status Changed

A vulnerability's severity or fix state changed — e.g. upgraded from MEDIUM to HIGH, or went from unfixed to fixed.
